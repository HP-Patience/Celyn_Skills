# nsfc-figure-prompts

为 NSFC 基金申请书生成 AI 科研绘图提示词。8 张图模板，7 轮迭代验证。

## 为什么需要这个

NSFC 申请书里，架构图、技术路线图、机制示意图是审稿人第一眼扫的东西。但你自己写 prompt 让 AI 画图，画出来的要么太花哨像创业 pitch deck，要么信息密度不够像课堂作业。

这个 skill 做的事：给你一套经过验证的 prompt 模板和设计规范，让 AI 画出的图看起来像**同一个基金本子里的一套图**——学术风、信息密度高、能用颜色讲清楚概念层次。

## 核心理念

> 虚线框不是装饰，是语义工具——它告诉审稿人"这些元素属于一组"。

- **只用三种形状**：圆角矩形（模块）、圆形（枢纽）、菱形（里程碑）
- **颜色携带含义**：蓝 = 认知层、绿 = 决策层、珊瑚 = 执行层
- **每张图底部深蓝总结条**：白字一句话结论，审稿人扫底部就能串起全文
- **纯白背景，无阴影无渐变无 3D**：学术审稿人不信任视觉装饰
- **每个模块必须标题 + 描述两层文字**：只有一个词标题 = 没写到位
- **箭头标注流向**：每条连接线旁写清"什么在流动"

## 用法

```
帮我写一张技术路线图的 AI 绘图 prompt
```

```
我要画一个三层递进架构图，帮生成 prompt
```

```
把这张机制示意图转成 AI 绘图 prompt
```

## 你会得到

每个 prompt 按此模板输出：

```
## 图N：标题

核心信息：[一句话，审稿人看图必须带走的结论]

布局方向：[纵向堆叠/中心环绕/三列并排/环形/上下组合/二维坐标]

[逐模块：位置 + 形状 + 配色 + 精确中文文字 + 虚线框分组]

[箭头和连接线，每条旁注小字说明流向]

底部深蓝总结条白字："[核心信息总结]"
```

## 支持的布局类型

| 布局 | 适用场景 | 关键词 |
|---|---|---|
| 纵向堆叠 | 分层架构 | "自上而下，层间箭头" |
| 中心环绕 | 枢纽辐射 | "中央圆形，外围模块环绕" |
| 左右三列 | 流程/I/O | "左中右三列，左→中→右流" |
| 环形 | 循环/闭环 | "顺时针排列，弧形箭头" |
| 上下组合 | 两个相关概念 | "===== 上半部分 ===== / 下半部分" |
| 二维空间 | 分区/矩阵 | "坐标系，横轴/纵轴，分区" |

## 适用场景

- 正在写 NSFC 或其他基金申请书
- 需要学术风格的架构图、技术路线图、机制示意图
- 想要一套视觉统一的 Figure Set
- 不知道怎么向 AI 描述学术插画需求

## 不适用

- 需要花哨视觉效果（渐变、阴影、3D）的场合——这是学术申请，不是融资 pitch deck
- 非学术场景的通用插图
- 已经有成熟绘图工具链（Figma/Illustrator 手绘）的场景
